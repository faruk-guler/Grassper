## ✨ Hakkında

**Grassper**, hız ve gizlilik odaklı, tamamen açık kaynaklı bir Flutter not alma uygulamasıdır. Material 3 tasarım dilini kullanarak hem şık hem de işlevsel bir deneyim sunar. Verileriniz tamamen yerel cihazınızda (Hive NoSQL) saklanır ve isterseniz güvenli bir şekilde yedeklenebilir.

## 🚀 Öne Çıkan Özellikler

- 📝 **Hızlı Not Alma:** Minimalist editör ile düşüncelerinizi anında kaydedin.
- 📌 **Sabitleme (Pin):** Önemli notlarınızı her zaman en üstte tutun.
- 🌗 **Dinamik Tema:** Dark ve Light mode desteği ile gözlerinizi yormayın.
- 📅 **Takvim Görünümü:** Notlarınızı tarihlere göre takip edin ve filtreleyin.
- 📦 **Arşiv & Çöp Kutusu:** Gereksiz notları arşive kaldırın veya çöp kutusuna atarak daha sonra geri yükleyin.
- 🔄 **İçe/Dışa Aktarma:** Verilerinizi JSON formatında yedekleyin veya başka bir cihaza taşıyın.
- 📧 **E-Posta Yedekleme:** Tek tıkla tüm notlarınızı güvenli bir e-posta yedeği olarak gönderin.
- 🎨 **Kişiselleştirme:** Profil fotoğrafı ekleyin ve hafta başlangıç gününü tercihinize göre ayarlayın.

## 🛠 Kullanılan Teknolojiler

- **Framework:** [Flutter](https://flutter.dev)
- **State Management:** [Provider](https://pub.dev/packages/provider)
- **Local Storage:** [Hive](https://pub.dev/packages/hive) (NoSQL)
- **Font:** [Google Fonts (Outfit)](https://fonts.google.com/specimen/Outfit)
- **Diğer Paketler:** `uuid`, `table_calendar`, `share_plus`, `image_picker`, `file_picker`.

## 📦 Kurulum ve Çalıştırma

Projeyi yerel makinenizde çalıştırmak için şu adımları izleyin:

1. Bu depoyu clonlayın:

   ```bash
   git clone https://github.com/kullanici-adi/grassper.git
   ```

2. Proje dizinine gidin:

   ```bash
   cd grassper
   ```

3. Bağımlılıkları yükleyin:

   ```bash
   flutter pub get
   ```

4. Uygulamayı çalıştırın:

   ```bash
   flutter run
   ```
